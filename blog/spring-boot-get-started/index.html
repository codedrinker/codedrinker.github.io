前言

最近 Spring Boot 如火如荼，于是也开始试试，不过上手真的太简单了。Idea非常方便的就可以创建，或者直接clone官方的例子https://spring.io/guides/gs/spring-boot/。通俗理解Spring Boot就是一个平台，让你非常便捷的构建和运行一个项目，并且他是基于组件化的，你想用什么直接拿来主义就可以了，话不多少直接上手。
